# 编程作业4
## 目标
* 实现一个魔方
## 环境
* Windows + Visual Studio
* OpenGL 3.3 + GLFW + GLAD + GLM
## 完成情况
* 实现了一个三阶魔方
## 操作说明
* 以6个数字对应魔方中的6种基本旋转：
  * 0：L
  * 1：R
  * 2：D
  * 3：U
  * 4：B
  * 5：F
